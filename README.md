# ksTFL Binaries

Pre-built binaries for the ksTFL package.

## Windows installation

```r
install.packages("ksTFL", repos = "https://<user>.github.io/ksTFL-release")
```

## Linux installation

Download the `.tar.gz` from the [releases/](releases/) folder, then:

```r
install.packages("ksTFL_<version>_R_x86_64-pc-linux-gnu.tar.gz", repos = NULL)
```
