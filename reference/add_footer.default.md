# Default method for add_footer

Default method for add_footer

## Usage

``` r
# Default S3 method
add_footer(spec, ...)
```

## Arguments

- spec:

  Spec object

- ...:

  Footer parts

  - Up to 3 positional character strings: left, center, right.

  - Use an empty string "" to represent an empty cell.

  - The `level` parameter can be used to replace an existing row instead
    of appending.

## Value

Error if no method found
